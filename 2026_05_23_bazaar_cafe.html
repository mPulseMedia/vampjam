About this page
• Source list at the top — add Google Drive shares, direct URLs, or filenames next to this page. Tap a row to load. The selected source is remembered per page.
• Pin to repo (📌): after adding a source locally, tap 📌 to commit it into sources/<PAGE_ID>.json via the GitHub API — then every visitor sees it. Pinned rows show 📍; tap to unpin. Requires a one-time PAT setup inside the "Audio source" panel.
• Player buttons: ▶/⏸ plays; −15s / +15s for fine seeking; −2m / +2m for big jumps. Click the bar to scrub. Spacebar = play/pause; T = tag.
• Tag row: ≡ drag handle (in Manual sort), time (click to seek), tiny −1s/+1s nudge the tag's saved time, inline label edit, 🔗 copies a direct link, ✕ deletes.
• Sort toggle: Time auto-sorts by timestamp. Manual respects your drag order. Dragging in Time mode auto-switches to Manual.
• Direct link formats: ?t=1h7m32s, ?t=13m22s, ?t=802, ?t=13:22. Also accepts #t=….
• Tags + sources save in localStorage. Use Export / Import to move between devices.
